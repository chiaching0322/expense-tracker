# Expense-tracker
利用Node.js Express 框架打造的簡易記帳網站。

## Features
- 使用者可以總攬所有支出清單
- 使用者可以依據 類別 ，篩選目前瀏覽支出清單
- 使用者可以看到目前清單上所有支出的總計金額
- 使用者可以先增一筆支出
- 使用者可以編輯支出的所有屬性
- 使用者可以刪除任何一筆支出

## Installation
- Clone the project 
```
git clone https://github.com/chiaching0322/expense-tracker.git
```
- Navigate to the project folder
```
cd expense-tracker
```
- Install npm 
```
install npm
```
- Seed
```
npm run seed
```
- Start the server
```
npm run develop
```
- Navigate to localhost:3000

## Prerequisites
- body-parser 1.19.0
- express 4.17.1
- express-handlebars 5.3.2
- mongoose 5.13.2
- bootstrap 4.3
- method-override 3.0.0
